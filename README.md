# Hot Scooby

Site desenvolvido para um food truck de lanches, com foco em praticar desenvolvimento Front-End utilizando HTML, CSS, JavaScript e Bootstrap. O projeto apresenta cardápio, informações do estabelecimento, mapa e conteúdos interativos.

## Autor
Nicoli Santos

## Funcionalidades
- Tela de pré-carregamento  
- Banner com slider automático  
- Animação numérica (counters)  
- Cardápio em PDF  
- Seção Sobre  
- Mapas incorporados (localidades fictícias)  
- Responsividade

## Tecnologias utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Bootstrap

## Objetivo do projeto
Projeto criado para fins educacionais, com foco em prática de estruturação de páginas web, responsividade e experiência do usuário.
